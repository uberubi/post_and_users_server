## Posts and Users App
  App that allows you to register, authentication, make/delete posts, like/comment on/delete your/other users posts.

## Technologies used
1. React
2. Node js
3. Apollo
4. GraphQL
5. Mongo DB
6. Semantic UI
7. Deploy: Netlify (client) + heroku (server)
